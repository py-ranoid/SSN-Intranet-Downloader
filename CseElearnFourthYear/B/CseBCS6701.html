Course Notes
S.NO Name of the topic Preclass Inclass
UNIT - I
INTRODUCTION
1 Security Services, Mechanisms, Attacks  
Network security model    
2 Classical Cryptography  
Symmetric cipher, Substitution, Permutation Ciphers    
3 FINITE FIELDS AND NUMBER THEORY  
Groups, Rings, Fields-Modular arithmetic, Euclid’s algorithm-Finite fields    
3 Prime numbers-Fermat’s and Euler’s theorem  
Testing for primality,Chinese remainder theorem, Discrete logarithms    
UNIT - II
BLOCK CIPHERS & PUBLIC KEY CRYPTOGRAPHY
1 Data Encryption Standard  
2 Block cipher principles, modes of operation, Triple DES  
3 Advanced Encryption Standard (AES)  
4 Blowfish, RC5 algorithm  
5 Public key cryptography: Principles of public key cryptosystems  
6 RSA algorithm  
7 Key management  
8 Diffie Hellman Key exchange  
9 Elliptic curve arithmetic, Elliptic curve cryptography  
UNIT - III
HASH FUNCTIONS AND DIGITAL SIGNATURES
1 Authentication requirement, Authentication function , MAC, Security of MAC  
2 Hash function, Security of hash function    
3 MD5, SHA 2    
4 Digital signature and authentication protocols  
5 DSS    
6 EI Gamal, Schnorr    
UNIT - IV
SECURITY PRACTICE & SYSTEM SECURITY
1 Authentication applications: Kerberos  
2 X.509 Authentication services  
3 SET  
3 Firewalls  
4 Trusted Systems  
5 Intruder Intrusion detection system  
6 Virus and related threats  
UNIT - V
SECURITY PRACTICE & SYSTEM SECURITY
1 E-mail Security: Pretty Good Privacy,S/MIME  
2 IPSecurity  
3 Web Security-SSL/TLS
