Course Objectives

  1. Learn the security issues in network layer and transport layer
  2. Be exposed to security issues of the application layer
  3. Learn computer forensics
  4. Be familiar with forensics tools
  5. Learn to analyze and validate forensics data

Course Outcomes (CO)

At the end of the semester, students are able to

  1. Discuss the security issues network layer and transport layer
  2. Apply security principles in the application layer
  3. Explain computer forensics
  4. Use forensics tools
  5. Analyze and validate forensics data
