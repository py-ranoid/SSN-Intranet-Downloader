Course Code & Name: CS6010 - SOCIAL NETWORK ANALYSIS

Course Handler: Dr. V.S. Felix Enigo

Syllabus:

Lesson Plan:

UNIT - I

  • Introduction to Social Network Analysis & Semantic Network

  • Evolution of Semantic Web

  • Development of Social Network Analysis

  • Key concepts and measures in SNA

  • Macro Structure of Social Networks

  • Social Capital - Concepts and Measures

  • Electronic Sources of Network Analysis

UNIT - II

  • Knowledge Representation using Ontologies

  • Resource Description Framework

  • Web Ontology Language (OWL)

  • State of Art in Network Data Representation

  • Ontological Representations

  • Conceptual Model

  • Aggregating, Reasoning and Advanced Representations

UNIT - III

  • Extracting Evolution of Web Communities from a Series of Web Archives

  • Detecting Communities in Social Networks

UNIT - IV

  • Understanding and Predicting Human Behavior in Social Networks

  • Trust in Online Environment

  • Privacy in Online Social Network

  • Attack Spectrum and Countermeasures

UNIT - V

  • Visualization

  • Node Link Diagrams

  • Applications of Social Network Analysis
