COURSE OBJECTIVES

  • The student should be made to:

  1. Understand how Grid computing helps in solving large scale scientific problems.
  2. Gain knowledge on the concept of virtualization that is fundamental to cloud computing.
  3. Learn how to program the grid and the cloud.
  4. Understand the security issues in the grid and the cloud environmen

Blooms Taxonomy

  • Remember -- K1
  • Understand -- K2
  • Apply -- K3
  • Analyze -- K4
  • Evaluate -- K5
  • Create -- K6

COURSE OUTCOMES

  • Students who successfully complete this course should be able to:

  1. Understand & Apply grid computing techniques to solve problems. [K3]
  2. Understand & Apply the concept of virtualization. [K3]
  3. Understand & Use the grid and cloud tool kits. [K3]
  4. Understand the security models in the grid and the cloud environment. [K2]
