COURSE OBJECTIVES

  • The student should be made to:

  1. Be exposed to tool kits for grid and cloud environment.
  2. Be familiar with developing web services/Applications in grid framework
  3. Learn to run virtual machines of different configuration.
  4. Learn to use Hadoop

Blooms Taxonomy

  • Remember -- K1
  • Understand -- K2
  • Apply -- K3
  • Analyze -- K4
  • Evaluate -- K5
  • Create -- K6

COURSE OUTCOMES

  • Students who successfully complete this course should be able to:

  1. Understand & Apply the concepts of Web Services [K3]
  2. Understand & Use the grid and cloud tool kits. [K2]
  3. Understand & Apply to develop applications on the Grid. [K3]
  4. Understand & Apply to develop applications on the Cloud. [K3]

Command to check if CPU supports Virtualization or Not

vmx – Intel VT-x, virtualization support enabled in BIOS.

svm – AMD SVM,virtualization enabled in BIOS.

]$ grep --color vmx /proc/cpuinfo

]$ cat /proc/cpuinfo| egrep "vmx|svm"
